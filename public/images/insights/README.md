# Insight Image Asset Guide

This folder contains the image assets used for the pop-up modals on the Insights page.

> **✅ CORRECT & FINAL FILE STRUCTURE:**
> This `insights` folder **MUST** be located inside `/public/images/`.
>
> - **Correct Path:** `[Your Project]/public/images/insights/data-lake.jpg`
> - **Incorrect Path:** `[Your Project]/images/insights/data-lake.jpg`

When referencing an image in the code, the path will be `/images/insights/data-lake.jpg`.

---

> **⚠️ CRITICAL: File Extension Must Be `.jpg`**
> All filenames listed below **MUST** use the `.jpg` extension.
> Using `.jpeg` or `.png` will result in an "Image not found" error. Please double-check every file.
