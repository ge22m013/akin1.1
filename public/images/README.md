# Image Asset Guide

This folder contains the main image assets for the website.

> **âœ… CORRECT & FINAL FILE STRUCTURE:**
> The `images` folder **MUST** be located inside the `/public` directory. The web server serves all files from `/public` as the root of the website.
>
> - **Correct Path:** `[Your Project]/public/images/platform-ui.jpg`
> - **Incorrect Path:** `[Your Project]/images/platform-ui.jpg`

When referencing an image in the code, use an absolute path from the root, like so: `<img src="/images/platform-ui.jpg" />`. The server automatically knows to look for this file inside `/public`.
