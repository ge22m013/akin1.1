# Partner Logo Asset Guide

This folder should contain the logo assets for our technology and strategic partners.

> **âœ… CORRECT & FINAL FILE STRUCTURE:**
> This `partners` folder **MUST** be located inside `/public/images/`.
>
> - **Correct Path:** `[Your Project]/public/images/partners/aws.png`
> - **Incorrect Path:** `[Your Project]/images/partners/aws.png`

When referencing a logo in the code, the path will be `/images/partners/aws.png`.
